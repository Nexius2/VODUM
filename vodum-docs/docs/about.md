# About VODUM

VODUM is a self-hosted Plex/Jellyfin user, subscription, access and policy
manager created by NeXius2. It is distributed under the MIT License.

- [GitHub repository](https://github.com/Nexius2/VODUM)
- [Discord community](https://discord.gg/5PU7TnegZt)
- [Documentation home](index.md)

The application About page displays the running version and platform details.
Include those values, relevant anonymized logs and clear reproduction steps when
requesting support.
